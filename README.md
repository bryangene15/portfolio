# Bryan Gene C. Hombre - Resume Website

This is a simple professional resume website created with HTML and CSS.

## Files

- `index.html` - Main resume website page
- `style.css` - Website styling

## How to Upload to GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html` and `style.css`.
3. Go to repository **Settings**.
4. Click **Pages**.
5. Under **Build and deployment**, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
6. Save and wait for GitHub to generate the website link.

## Before Publishing

Replace these placeholders inside `index.html`:

- `[Insert Number]`
- `[Insert Email]`
